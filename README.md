__Телеграм бот по продаже подписок которую вы добавите в базу данных sql в sqlite . 
Можно вписать несколько категорий подписки я почему то назвал это категориями хотя это варианты ну да ладно. 
Кому лень кодить можете взять и перекодить под себя__

Я не добавлял админ панель и генерацию ссылок на канал / группу. Можете после оплаты добавить сообщение с вашим каналом...

В боте все можно изменить под себя. В
базе данных (sqlite) можно изменить: цену, описание, название и время (только в месяцах) , там есть две таблицы - юзеры и категории подписок. По умолчанию в боте стоит 1 звезда в оплате..

Всем спасибо!


Импорты:
1. aiogram
2. sqlalchemy
3. python-dateutil наскока помню
4. greenlet
5. python-dotenv

В файл .env вставляем свой токен бота без кавычек
